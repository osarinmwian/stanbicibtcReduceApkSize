# constant Box

# constant Text

Custom Text component with type checked layout stylings and props including typography. All styles can be passed as props rather than using the StyleSheet API, and we can select a variant with predefined styles.

Includes all the props that are available in the native Text component. Fully themeable.

See:
  • https://github.com/Shopify/restyle#text
